## 版本
- Java：JDK 11

## 组件
### 服务注册中心
服务注册中心使用Consul，可以去[官网下载](https://www.consul.io/downloads.html)
使用默认端口号，Web界面端口是8500。
### service-user
用户微服务：
- 服务名：service-user
